# Section 16：環境傷害 (Environmental Injuries)

---

## Cold Injuries

### 非凍結性冷傷害

**Trench Foot：**
- 長時間暴露於潮濕、低溫（>0°C）
- 早期：tingling、numbness、pale、mottled、pulseless
- 後期：hyperemic phase、burning pain、edema、bullae
- 治療：supportive、keep clean/warm/dry/elevated

**Chilblains (Pernio)：**
- 長期間歇性暴露於潮濕、非凍結溫度
- 部位：feet、hands、ears、lower legs
- 表現：erythema、cyanosis、edema、pruritus、burning
- 治療：rewarm、nifedipine 20mg tid、topical corticosteroids

**Cold Urticaria：**
- 對冷空氣/水的過敏反應
- 可導致致命 anaphylaxis
- 治療：antihistamines (H1)、epinephrine autoinjector（如果有 anaphylaxis 歷史）

## Frostbite (凍結性傷害)

### 分級

| Degree | 臨床表現 |
|--------|----------|
| **First degree (frostnip)** | Numbness、erythema、mild edema、desquamation |
| **Second degree** | Full-thickness skin freezing、clear blisters、substantial edema |
| **Third degree** | Hemorrhagic blisters、skin necrosis、blue-gray discoloration |
| **Fourth degree** | 深層組織受損（muscle、bone、tendon）、mummified eschar |

### 風險因素

| 類別 | 因素 |
|------|------|
| **環境** | 溫度、風速、濕度、海拔 |
| **個人** | Age、gender、alcohol、smoking、clothing |
| **健康** | Raynaud's、diabetes、peripheral vascular disease、previous cold injury |

### 病理生理

- **Endothelial damage** → arachidonic acid cascade → vasoconstriction → thrombosis → ischemia/necrosis
- **組織分區**：zone of coagulation（不可逆）、zone of stasis（可能可逆）、zone of hyperemia（恢復良好）

### 治療

**Field care：**
- 移除濕/緊身衣物
- 覆蓋乾衣物
- **不要  rewarm 直到避免 refreezing**
- 給止痛藥

**ED management：**
- **Rapid rewarming**：水溫 37-39°C，20-30 分鐘直到組織軟化、發紅
- **Parenteral opiates**（rewarming 很痛）
- **Topical aloe vera** q6h（對抗 arachidonic acid cascade）
- **Ibuprofen** 12 mg/kg/day divided doses

**Blister management：**
- **Clear blisters**：有爭議；blister fluid 含 thromboxane/prostaglandins
- **Hemorrhagic blisters**：不要 debride

**Optional：**
- Topical bacitracin 或 penicillin G 500,000U IV q6h × 48-72h
- Tetanus prophylaxis

**不建議：**
- Early surgery（需要數週才能判斷組織存活）
- Heparin（無明確效益）
- Early sympathectomy

## Hypothermia

### 定義

- **Accidental hypothermia**：core temperature <35°C (<95°F)
- 分為 primary（單純環境暴露）和 secondary（散熱增加或產熱減少）

### 分期

| Stage | 臨床表現 | 溫度 | 治療 |
|-------|----------|------|------|
| **HT I (Mild)** | Conscious、shivering | 35-32°C | Warm environment、clothing、drinks |
| **HT II (Moderate)** | Impaired consciousness | <32-28°C | Active external rewarming、cardiac monitoring |
| **HT III (Severe)** | Unconscious、vital signs present | <28°C | HT II + airway management、consider ECMO/CPB |
| **HT IV** | Vital signs absent | - | CPR、transport to ECMO/CPB center |

### 生理變化

| 系統 | 變化 |
|------|------|
| **CNS** | Impaired judgment → ataxia → unconscious → areflexia |
| **CV** | Initial tachycardia → bradycardia、hypotension、arrhythmias |
| **Resp** | Tachypnea → hypoventilation → respiratory arrest |
| **Renal** | Cold diuresis（vasoconstriction-induced hypervolemia） |
| **Coag** | Coagulopathy（<34°C）、hypercoagulable（部分患者） |

### ECG 變化

- **Osborn J waves**：經典變化，通常 <32°C
- 可出現任何心律不整
- Atrial fibrillation/flutter 是預期中的，不是立即治療的適應症

### 治療原則

**Mild hypothermia (HT I)：**
- Warm environment
- Warm drinks
- Active movement

**Moderate/Severe (HT II-III)：**
- **Active external rewarming**：warm blankets、forced-air warming
- **Warm IV fluids**
- **Cardiac monitoring**
- **Minimal movements**（避免 arrhythmia）

**HT IV / Cardiac arrest：**
- CPR
- Epinephrine（可用，但 metabolism 慢）
- **Transport to ECMO/CPB center**（如果可用）
- **Consider ECMO/CPB** for cardiac instability refractory to medical management

**ECMO/CPB 適應症：**
- HT III with cardiac instability
- HT IV
- Cannot tolerate low cardiac output

### 特殊考量

**"Rescue collapse"：**
- 在 extrication/transport/treatment 時可能發生 cardiac arrest
- 原因：cold myocardium irritability

**"Afterdrop"：**
- 離開冷環境後 core temperature 可能繼續下降
- 健康能 shiver 的患者通常能 self-rewarm

**Iatrogenic hypothermia：**
- 常見於創傷患者
- 可導致 coagulopathy 和 mortality 增加
- 使用 fluid warmers 和保溫

## Electrical Injuries

### 病理生理

| 效應 | 說明 |
|------|------|
| **Direct tissue damage** | 電流通過組織的熱效應 |
| **Muscle contraction** | 可導致骨折、關節脫臼 |
| **Cardiac** | VF、asystole、arrhythmias |
| **血管** | Vasospasm、thrombosis |
| **神經** | Paralysis、paresthesias |

### 臨床表現

| 系統 | 表現 |
|------|------|
| **Skin** | Burns（entry/exit points）、thermal burns |
| **Cardiac** | Arrhythmias、MI |
| **神經** | Paralysis、paresthesias、SCI |
| ** Renal** | Myoglobinuria（rhabdomyolysis） |

### 治療

| 治療 | 說明 |
|------|------|
| **BLS/ACLS** | VF → defibrillation |
| **Fluid resuscitation** | 根據燒傷面積計算（Parkland formula）|
| **Tetanus** | prophylaxis |
| **Monitor** | ECG 24h（如果有任何心臟症狀） |

## Lightning Injuries

### 臨床特徵

| 表現 | 說明 |
|------|------|
| **Cardiac** | Immediate cardiac arrest（asystole 或 VF）|
| **Neurologic** | LOC、paralysis、SCI |
| ** cutaneous** | Burns、fern-like patterns（Lichtenberg figures）|
| **Auditory** | Tympanic membrane rupture |

### 治療

- **BLS/ACLS**
- **Tympanic membrane rupture**：是預期中的（不是排除標準）
- **Admission**：所有 lightning strike 患者

## Diving Emergencies

### Decompression Illness (DCI)

**原因：** 氮氣在減壓時形成氣泡

**類型：**
| 類型 | 表現 |
|------|------|
| **DCS Type I** | Joint pain、skin manifestations、lymphedema |
| **DCS Type II** | 神經、心臟、肺部症狀 |
| **AGE** | Cerebral ischemia、spinal cord injury |

### 治療

| 治療 | 說明 |
|------|------|
| **100% O2** | 減少氮氣負荷 |
| **IV fluids** | - |
| **Recompression** | 必需；consult diving medicine |
| **Transport** | 盡快送到 recompression chamber |

### 減壓錶/飛航

- 送出後 24 小時內避免飛航
- 如果需要飛航，使用 100% O2 在飛機上

## 高山症 (Altitude Illness)

### 急性高山症 (AMS)

| 症狀 | 說明 |
|------|------|
| **頭痛** | 典型的 |
| **GI** | Nausea、vomiting、anorexia |
| **CNS** | Fatigue、dizziness、sleep disturbance |

### 高山腦水腫 (HACE)

- 嚴重的 AMS + altered mental status、ataxia
- 可能致命

### 高山肺水腫 (HAPE)

- 非心因性 pulmonary edema
- 症狀：dyspnea、cough、pink frothy sputum

### 治療

| 疾病 | 治療 |
|------|------|
| **AMS** | Descend、acetazolamide、oxygen |
| **HACE** | Immediate descent、dexamethasone |
| **HAPE** | Immediate descent、nifedipine、oxygen、portable hyperbaric chamber |

### 預防

- **Acetazolamide** 125-250mg bid
- **Dexamethasone** 2mg q6h 或 4mg q12h
- **Altitude acclimatization**：慢慢上升

## 輻射傷害

### 急性輻射症候群 (ARS)

| 階段 | 時間 | 表現 |
|------|------|------|
| **Prodromal** | Hours | Nausea、vomiting、diarrhea |
| **Latent** | Days-weeks | Asymptomatic |
| **Manifest** | 取決於劑量 | Bone marrow（pancytopenia）、GI（bleeding、dehydration）、CV/CNS（death） |

### 治療

| 治療 | 說明 |
|------|------|
| **Supportive** | Fluid、electrolytes、transfusions |
| **Filgrastim (G-CSF)** | Bone marrow suppression |
| **Prophylaxis** | Antibiotics、antifungals |
| **Decontamination** | 如果有 external contamination |

### Decorporation agents

| 藥物 | 用途 |
|------|------|
| **Prussian blue** | Cesium、thallium |
| **DTPA (Ca-DTPA 或 Zn-DTPA)** | Plutonium、americium、curium |
| **Sodium bicarbonate** | Uranium |
