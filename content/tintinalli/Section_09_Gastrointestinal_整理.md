# Section 09：胃腸道疾病

## 急性腹痛 (Acute Abdominal Pain)

### 疼痛機制

| 類型 | 特點 |
|------|------|
| **Visceral pain** | 模糊、痙攣性、定位差 |
| **Parietal pain** | 局部、可定位、壓痛 |
| **Referred pain** | 遠處器官 |

### 胚胎來源與疼痛位置

| 胚胎來源 | 疼痛位置 |
|---------|---------|
| **Foregut** | 上腹部 |
| **Midgut** | 臍周 |
| **Hindgut** | 恥骨上 |

### 高風險族群

- **年齡 extremes**：老年人、幼兒
- **免疫抑制**：HIV、化療、類固醇
- **懷孕**

### 危及生命徵象

- 姿勢性低血壓（站立時 HR 增加 ≥30/min）
- 腹膜徵象：rigidity、rebound tenderness
- 休克

### 理學檢查

| 檢查 | 意義 |
|------|------|
| **Decreased bowel sounds** | Ileus、腸阻塞、peritonitis |
| **Fluid wave** | Ascites |

### 診斷檢驗

| 檢驗 | 適用情況 |
|------|---------|
| **Amylase/Lipase** | 胰臟炎（lipase 更特異） |
| **Lactate** | 腸繫膜缺血 |
| **β-hCG** | 所有生育年齡女性 |

### 影像學

| 檢查 | 適用情況 |
|------|---------|
| **Plain X-ray** | 腸阻塞、游離氣體 |
| **Ultrasound** | 膽囊疾病、腎積水 |
| **CT with IV contrast** | 闌尾炎、憩室炎、胰臟炎 |

### 常見診斷（按位置）

| 位置 | 診斷 |
|------|------|
| **瀰漫性** | Aortic aneurysm、mesenteric ischemia、bowel obstruction、pancreatitis |
| **RUQ** | Cholecystitis、biliary colic、cholangitis |
| **RLQ** | Appendicitis、Crohn's disease |
| **LUQ** | Gastric ulcer、pancreatitis、splenic rupture |
| **LLQ** | Diverticulitis、ovarian torsion |

### 闌尾炎特點

- **遷移性疼痛**：Periumbilical → RLQ
- **Pain before vomiting**
- **Anorexia**：84% sensitivity
- **CT 敏感度**：93-100%

### 膽囊炎特點

- **Murphy sign (+)**：吸氣時 RUQ 壓痛停止
- **Jaundice**：提示 CBD obstruction

### 憩室炎特點

- **LLQ 痛**（85% sigmoid）
- **CT 敏感度**：93-100%

### 腸繫膜缺血特點

- **Pain out of proportion to physical findings**
- Atrial fibrillation（常見原因）
- **Lactate**：75-90% 敏感度

### 治療原則

- **Opioids 不會掩蓋腹部徵象**，不應延遲給予
- **抗生素**：根據感染來源（社區性 vs. 膽道性）

## 胃腸道出血 (GI Bleeding)

### 上消化道出血

| 病因 | 說明 |
|------|------|
| **Varices** | 門脈高壓；紅色徵兆表示再出血風險 |
| **PUD** | 最常見；Hp + NSAIDs + aspirin |
| **Mallory-Weiss tear** | 嘔吐後 |
| **Esophagitis** | GERD 並發症 |

### 下消化道出血

| 病因 | 說明 |
|------|------|
| **Diverticulosis** | 最常見（40%）|
| **Angiodysplasia** | 老年人；主動脈狹窄相關 |
| **Colorectal cancer** | 需排除 |
| **Hemorrhoids** | 排除其他後考慮 |

### 評估

| 項目 | 說明 |
|------|------|
| **Hemodynamics** | 姿勢性 HR/BP 變化 |
| **Nasogastric tube** | 區分上下出血 |
| **EGD** | Gold standard for 上消化道 |
| **Colonoscopy** | Gold standard for 下消化道 |

### 風險分層

| 分數 | 內容 |
|------|------|
| **GBS** | 敗血症、意識、BUN、紅血球输注、嘔血、肝病 |
| **Child-Pugh** | Albumin、INR、Bilirubin、Ascites、Encephalopathy |

### 治療

| 項目 | 治療 |
|------|------|
| **Varices** | Octreotide、antibiotics、banding |
| **PUD** | PPI、HP eradication |
| **Diverticular** |  Colonoscopy 後 clip/coagulation |

## 肝膽疾病

### 膽結石

- **風險因素**：Female、Fertile、Fat、Fifteen（快速減重）、Forty
- **膽囊結石**：超聲（sensitivity >95%）

### 急性膽囊炎

| 項目 | 說明 |
|------|------|
| **診斷** | Charcot's triad（fever、RUQ pain、jaundice）|
| **Murphy sign (+)** | 吸氣時 RUQ 壓痛停止 |
| **超聲** | Gallstones、wall thickening >4mm、pericholecystic fluid |

### 急性膽管炎

| 項目 | 說明 |
|------|------|
| **Charcot's triad** | Fever、RUQ pain、jaundice |
| **Reynold's pentad** | + altered mental status、hypotension |
| **治療** | Antibiotics + ERCP drainage |

### 胰臟炎

| 項目 | 說明 |
|------|------|
| **診斷** | 2 of 3：characteristic pain、lipase >3x ULN、imaging |
| **APACHE II >8** | 預測 severe pancreatitis |
| **壞死性胰臟炎** | CT with IV contrast（48-72h 後）|

### 治療

| 項目 | 說明 |
|------|------|
| **液體復甦** | LR 250-500 mL/h（目標 HR <120）|
| **疼痛控制** | Opioids |
| **ERCP** | 如有 cholangitis 或 biliary obstruction |
| **NPWT** | 如有 walled-off necrosis |

## 肛門直腸疾病

### 痔瘡

| 類型 | 說明 |
|------|------|
| **內痔** | 齒狀線以上；无痛性出血 |
| **外痔** | 齒狀線以下；疼痛、血栓 |

### 治療

| 情況 | 治療 |
|------|------|
| **症狀性內痔** | Rubber band ligation |
| **血栓性外痔** | 切除血栓 |

### 肛裂

| 項目 | 說明 |
|------|------|
| **特點** | 排便時疼痛 + 鮮血 |
| **位置** | Posterior midline（90%）|
| **治療** | GTN ointment、nifedipine、botox |

## 腹壁疝氣

| 類型 | 說明 |
|------|------|
| **Inguinal hernia** | 最常見；indirect vs. direct |
| **Femoral hernia** | 女性多；絞窄風險高 |
| **Umbilical hernia** | 成人需手術 |

---

## 重點摘要

- **闌尾炎**：遷移性疼痛 + pain before vomiting + CT 敏感度 93-100%
- **膽囊炎**：Murphy sign (+) + 超聲
- **膽管炎**：Charcot's triad + ERCP drainage
- **胰臟炎**：2 of 3 診斷標準 + LR 大量輸液
- **腸繫膜缺血**：Pain out of proportion + lactate
- **GI bleed**：GBS 分數 + 早期內視鏡
- **Varices**：Octreotide + antibiotics + banding
