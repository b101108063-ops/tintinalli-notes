# Section 26：特殊情況 (Special Situations)

---

## 注射藥物使用者 (IDU)

### 常見併發症

| 系統 | 併發症 |
|------|--------|
| **感染** | HIV、肝炎、蜂窩組織炎、心內膜炎 |
| **肺部** | 肺炎、膿胸、氣胸 |
| **心臟** | 心內膜炎 |
| **神經** | 腦膜炎、腦膿瘡、脊柱膿瘡 |
| **其他** | 創傷、藥物過量 |

---

## 發燒評估

| 原因 | 說明 |
|------|------|
| **感染** | 最常見（>2/3）|
| **藥物反應** | 急性毒性反應、摻雜物反應 |
| **戒斷** | Benzodiazepine、barbiturate、heroin |
| **Cotton fever** | 注射後數小時內發生、flu-like、自限性 |

---

## 注射藥物相關感染

### 皮膚/軟組織感染

| 感染 | 說明 |
|------|------|
| **Cellulitis** | 紅腫熱痛 |
| **Abscess** | 膿瘡 |
| **Necrotizing fasciitis** | 緊急！ |
| **Botulism** | 傷口型、進展性麻痺 |

### 心內膜炎 (Endocarditis)

| 類型 | 病原 |
|------|------|
| **Right-sided (tricuspid)** | Staph aureus（最常見）|
| **Left-sided** | 多种病原 |
| **Native valve** | Viridans streptococci、Staph aureus |
| **Prosthetic** | 多重病原 |

**Duke Criteria：**
| 主要 | 次要 |
|------|------|
| 陽性血培養 ×2 | 靜脈藥癮 |
| 心內膜受損證據 | 傾向 |
| | 超聲波陽性所見 |

## 肺部併發症

| 疾病 | 說明 |
|------|------|
| **Pneumonia** | 典型/非典型 |
| **Aspiration** | 意識減退時 |
| **Pneumothorax** | "Pocket shooting"（鎖骨上注射）|
| **Septic pulmonary emboli** | right-sided endocarditis |
| **Talc lung** | 摻雜物引起的間質浸潤 |
| **Noncardiogenic pulmonary edema** | Heroin、cocaine |

---

## 神經併發症

| 疾病 | 說明 |
|------|------|
| **Epidural abscess** | 背痛 + 發燒 + 神經學缺陷 |
| **Bacterial meningitis** | Staph aureus 最常見 |
| **Brain abscess** | 来自心內膜炎栓塞 |
| **CNS opportunistic infection** | Toxoplasma（CD4 <100）|
| **Tetanus** | 傷口感染、肌肉痙攣 |

---

## 腸胃道/肝膽

| 疾病 | 說明 |
|------|------|
| **Viral hepatitis** | HAV、HBV、HCV |
| **Alcoholic hepatitis** | 合併酒精使用 |
| **Biliary disease** | 囊性疾病 |
| **Pancreatitis** | 藥物引起 |

---

## 眼科

| 疾病 | 說明 |
|------|------|
| **Herpes zoster** | 眼帶狀皰疹 |
| **CMV retinitis** | HIV 患者、CD4 <50 |
| **Toxoplasma retinitis** | HIV 患者 |
| **KS** | Kaposi sarcoma（眼瞼/結膜）|

---

## 腎臟/泌尿

| 疾病 | 說明 |
|------|------|
| **Glomerulonephritis** | HBV、HCV相關 |
| **Amyloidosis** | 長期注射 |
| **Renal abscess** | 栓塞性 |

---

## 肌肉骨骼

| 疾病 | 說明 |
|------|------|
| **Osteomyelitis** | 椎體、注射部位 |
| **Septic arthritis** | 鄰近感染擴散 |
| **Myositis/Fasciitis** | 嚴重軟組織感染 |

---

## 藥物過量

### Opioid Overdose

| 徵象 | 治療 |
|------|------|
| **Miosis、respiratory depression、coma** | **Naloxone** 0.4-2 mg IV |
| **Triad** | 必要時重複 |

### Cocaine/Amphetamine

| 徵象 | 治療 |
|------|------|
| **高血壓、心動過速、幻覺、癲癇** | Benzodiazepine |
| **胸痛** | 評估心臟 |
| **高熱** | 降溫、benzodiazepine |

---

## 戒斷症狀

### Opioid Withdrawal

| 徵象 | 治療 |
|------|------|
| **Mydriasis、lacrimation、rhinorrhea、piloerection、diaphoresis、diarrhea、yawning** | 支持性、Buprenorphine、Methadone |

### Alcohol/Benzodiazepine Withdrawal

| 徵象 | 治療 |
|------|------|
| **顫抖、焦慮、癲癇、意識混亂** | Benzodiazepine |

---

## 特殊考量

| 考量 | 說明 |
|------|------|
| **隱私** | 藥癮者可能擔心通報 |
| **社會因素** | 無家可歸、營養不良 |
| **精神疾病** | 合併憂鬱、焦慮、精神病 |
| **跟蹤困難** | 需確保後續照護 |

---

## 處理原則

| 步驟 | 內容 |
|------|------|
| **1. 評估 ABCD** | 保持氣道、呼吸、循環 |
| **2. 詳細病史** | 藥物種類、使用方式、注射部位 |
| **3. 身體檢查** | 注射痕跡、發燒、心臟雜音 |
| **4. 實驗室** | 血培養、藥物篩檢、肝炎/HIV |
| **5. 影像** | CXR、超聲波、CT |
| **6. 治療** | 抗生素、解毒劑 |
| **7. 轉介** | 藥癮治療、社會資源 |

---

## 門診追蹤考量

| 因素 | 說明 |
|------|------|
| **跟蹤能力** | 確保可靠 |
| **社會支持** | 家庭、庇護所 |
| **併發症** | 心內膜炎、敗血症需住院 |
| **注射痕跡感染** | 可門診治療 |

---

## 孕婦與藥物

| 考量 | 說明 |
|------|------|
| **Narcan** | 孕婦可用、不危害胎兒 |
| **戒斷** | 增加胎兒風險 |
| **Methadone/Buprenorphine** | 孕婦的替代治療 |
| **新生兒戒斷** | 嬰兒可能需要治療 |

---

## 戒毒治療選項

| 治療 | 說明 |
|------|------|
| **Methadone** | Long-acting opioid agonist |
| **Buprenorphine** | Partial agonist、可門診開立 |
| **Naltrexone** | Opioid antagonist |
| **Behavioral therapy** | 心理治療 |

---

## 公共衛生考量

| 考量 | 說明 |
|------|------|
| **通報** | 法定傳染病通報 |
| **針具交換** | 減少傷害策略 |
| **PrEP** | HIV 預防 |
| **Naloxone 分發** | 社群層級解毒劑 |
