# Section 03：復甦

## Sudden Cardiac Death

### 流行病學

- 美國每年約 **382,800** 成人發生 out-of-hospital cardiac arrest
- EMS-treated survival：**11.4%**；overall survival：**6.8%**
- **80%** 發生於家中；公共場所存活率較高（VF 發生率較高）
- 最低社會經濟階層 SCD 發生率高 **30-80%**
- 死亡高峰：醒後數小時（交感神經刺激增加）
- 兩年齡高峰：嬰兒期（SIDS）、**>45-50 歲（60% 為男性）**

### 病因

| 病因 | 比例 |
|------|------|
| **冠狀動脈疾病**（常為未診斷） | **80%** |
| **心肌病變** | 10-15% |
| **其他**（channelopathies、瓣膜疾病、先天異常） | 5-10% |

- **冠狀動脈疾病**：80% 驗屍發現 atherosclerosis；70-80% coronary angiography 發現；1/3 有 acute plaque rupture
- **嚴重左心室功能障礙**：EF ≤35% 為 ICD 主要候選
- **心肌病變**：LV 擴張 → 去極化/復極化分散 → circus movement reentry → VT/VF
- **肥厚性心肌病（HCM）**：年輕運動員最常見心血管死因（1/3）；每年 SCD risk ~**1%/year**
- **ARVC**：遺傳性右心病變；T-wave inversion in V1-3；ICD 為首選

### 先天性心臟病相關 SCD

- 左心阻塞性病變（主動脈瓣狹窄、主動脈窄縮）、發紺型缺損（Fallot 四重症、Ebstein's anomaly）
- 最大風險在運動時；50% 由 VF 造成

### Hereditary Channelopathies

| 疾病 | ECG 特徵 | 風險 |
|------|----------|------|
| **Brugada syndrome** | V1-3 downsloping ST elevation + RBBB pattern | 40-60% life-threatening VA；常見於東南亞 |
| **Early repolarization** | J wave + upsloping ST elevation（lateral/inferior） | 通常良性；>1%成人；運動員可達10% |
| **Long QT syndrome** | QTc 延長 | Torsade de pointes → VF；Bazett formula：QTc = QTm/√RR |
| **Short QT syndrome** | QTc <0.34s | AF、VT/VF、SCD |
| **CPVT** | 正常（運動/stress 誘發） | 50% initially diagnosed as epilepsy |

### 休克類型

| 類型 | 特色 |
|------|------|
| **Hypovolemic** | 出血、脫水；冷、濕、蒼白；CVP 低 |
| **Cardiogenic** | 心臟功能障礙；冷、濕；CVP 高；心臟衰竭證據 |
| **Obstructive** | PE、心包填塞、張力性氣胸；冷、濕；CVP 高 |
| **Distributive（Septic）** | 暖、低外週阻力；晚期冰冷；CVP 低 |
| **Neurogenic** | 創傷性脊髓損傷；溫暖、乾；心動過緩 |

### 休克評估與處理

- **MAP** = CO × SVR；目標 MAP >65 mmHg
- **乳酸**：>2 mmol/L 為組織灌流不足；目標 <2
- **ScvO2**（central venous O2 saturation）：>70%
- **尿量**：>0.5 mL/kg/hr
- Fluid resuscitation：crystalloid（lactated Ringer's 或 normal saline）
- **Vasopressors**：Norepinephrine（首選）、Dopamine、Epinephrine
- **Inotropes**：Dobutamine（增加 CO）、Milrinone（PDE-3 inhibitor）
- **VA-ECMO**：難復甦心因性休克

### 創傷性休克復甦

- **TASH score**（Trauma Associated Severe Hemorrhage）：預測 massive transfusion
- **MTP**（Massive Transfusion Protocol）：1:1:1（PRBC:FFP:Platelet）
- **Tranexamic acid**（TXA）：<3h 內 1g IV（第一小時）+ 1g（接下來 8h）
- **Permissive hypotension**：目標 SBP 80-90 mmHg（無頭部創傷）
- **REBOA**（Resuscitative Endovascular Balloon Occlusion of the Aorta）：用於不可壓迫出血
- **目標**：控制出血 → 維持組織灌流 → 防止低體溫/酸中毒/凝血病變三角

### 過敏性反應（Anaphylaxis）

- **診斷標準**（NIAID）：3 個器官系統 involve 或已知 allergen exposure + 2 個系統
- 皮膚症狀（90%）：蕁麻疹、血管性水腫、紅斑
- 呼吸（70%）：支氣管痙攣、喉頭水腫、喘鳴
- 心血管（45%）：低血壓、心搏過速、血管崩潰
- GI（30%）：腹痛、嘔吐、腹瀉

### 過敏性反應治療

- **Epinephrine**（IM）：1:1000，**0.3-0.5 mg**（成人）；0.01 mg/kg（兒童，max 0.3 mg）
- 位置：大腿外側（可重複每 5-15 分鐘）
- **IV fluids**：2-4 L crystalloid
- **Antihistamines**：Diphenhydramine 25-50 mg IV + Famotidine 20 mg IV
- **Corticosteroids**：Methylprednisolone 125 mg IV 或 Dexamethasone 10 mg IV
- **Glucagon**：1-5 mg IV（beta-blocker 患者）
- **Albuterol**：2.5-5 mg nebulized（持續支氣管痙攣）
- **Norepinephrine**：若持續低血壓（加強血管收縮）

### 酸鹼失衡

| 失衡 | 原因 | 特色 |
|------|------|------|
| **MA**（ Metabolic acidosis） | 乳酸（組織灌流不足）、 ketones、renal failure、toxins | HCO3 下降；Pco2 代償性下降 |
| **MAl**（Metabolic alkalosis） | 嘔吐、胃吸引、diuretics、鹼過量 | HCO3 上升 |
| **RA**（Respiratory acidosis） | 呼吸抑制、COPD 急性惡化、氣道阻塞 | Pco2 上升；HCO3 代償性上升 |
| **RAl**（Respiratory alkalosis） | 焦慮、過度換氣、發燒、藥物、肺栓塞 | Pco2 下降 |

### 補償公式

- **MA**：預期 Pco2 = 1.5 × HCO3 + 8（±2）
- **MAl**：預期 HCO3 上升 10 = Pco2 上升 5-7 mmHg
- **RA**：急性：HCO3 上升 1 = Pco2 上升 1；慢性：HCO3 上升 10 = Pco2 上升 7
- **RAl**：急性：HCO3 下降 2 = Pco2 下降 5；慢性：HCO3 下降 5 = Pco2 下降 10

### 血氣分析判讀步驟

1. 看 pH：<7.35 酸中毒；>7.45 鹼中毒
2. 看 Pco2 vs HCO3：哪個與 pH 方向一致（primary disorder）
3. 計算補償是否適當
4. 判斷是否為單純或混合失衡
5. 計算 Anion gap = Na - (Cl + HCO3)；正常 8-12
   - **高陰離子隙 MA**：Lactic acidosis、ketoacidosis、toxins（methanol、uremia、propylene glycol、 metformin）
   - **正常陰離子隙 MA（MUDPIES）**：慢性腎病、Addison's、dilutional、GI loss、輸入大量 saline、renal tubular acidosis

### 液體與電解質

| 項目 | 正常值 | 異常及時處置 |
|------|--------|--------------|
| **Na** | 136-145 mEq/L | <120：Hypertonic saline；>160：Free water |
| **K** | 3.5-5.0 mEq/L | <2.5：KCl；>6.5：Calcium、insulin/glucose、albuterol、Kayexalate |
| **Ca**（total）| 8.5-10.5 mg/dL | 低：MgSO4、CaCl2；心臟手術後監測 |
| **Mg** | 1.5-2.5 mg/dL | <1.5：替補；心臟手術後、Torsades、TOCS 警訊 |
| **Phosphate** | 2.5-4.5 mg/dL | 低：rebound after starvation、TPN |

### 心律不整

| 類型 | 特色 |
|------|------|
| **VT（Ventricular tachycardia）** | ≥3 consecutive PVCs；QRS >0.12s；穩定 vs. 不穩定 |
| **VF（Ventricular fibrillation）** | 無效心室收縮；需立即除顫 |
| **Asystole** | 完全無心臟活動；很少存活 |
| **PEA（Pulseless electrical activity）** | 有節律但無脈搏；需找出可逆原因 |
| **SVT** | >150 bpm；SVT with aberrancy vs. VT 鑑別重要 |

### 抗心律不整藥物

| 藥物 | 劑量 | 適應症 |
|------|------|--------|
| **Adenosine** | 6mg → 12mg（第二劑）| SVT diagnosis/終止；AV node reentrant |
| **Amiodarone** | 150mg IV（10min）→ 1mg/min | VT/VF、VF；wide-complex tachycardia |
| **Lidocaine** | 1-1.5 mg/kg IV | VT/VF（第二線）；窄-complex SVT |
| **Procainamide** | 20-50 mg/min IV | SVT、VT、WPW；QRS 增寬 >50% 需停藥 |
| **Esmolol** | 5-100 mcg/kg/min | SVT、高血壓、aortic dissection |
| **Verapamil/Diltiazem** | V: 5-10mg IV；D: 15-20mg/hr | SVT、AF with RVR |
| **Atropine** | 0.5 mg IV（可重複，最大 3mg）| Sympathetic/bradycardia、PEA with bradycardia |

### 電擊整流（Cardioversion）與除顫

| 情境 | 能量 |
|------|------|
| **同步電擊（Synchronized cardioversion）** | SVT/AF：100-200 J（biphasic）；VT（稳定）：100 J |
| **VF/無脈搏 VT** | 120-200 J（biphasic）；360 J（monophasic）|
| **PEA/Asystole** | 無法電擊；專注 CPR + Epinephrine |

### 升壓與強心藥物

| 藥物 | 劑量 | 作用 |
|------|------|------|
| **Norepinephrine** | 0.05-0.5 mcg/kg/min | α1+β1：強血管收縮（升血壓首選）|
| **Epinephrine** | 0.01-0.5 mcg/kg/min | β1+β2+α1：心臟復甦、CPR、過敏 |
| **Dopamine** | 2-20 mcg/kg/min | 低劑量：D1；中劑量：β1；高劑量：α1 |
| **Dobutamine** | 2-20 mcg/kg/min | β1>β2：增加心肌收縮力（心因性休克）|
| **Milrinone** | 0.375-0.75 mcg/kg/min | PDE-3 inhibitor：增加 cAMP；心因性休克、PAH |
| **Vasopressin** | 0.03-0.04 units/min | V1：純血管收縮；不受 β-blocker 影響 |
| **Dobutamine** | 2-20 mcg/kg/min | β1>β2：強心（心因性休克）|
| **Nitroglycerin** | 5-200 mcg/min | Venous dilator；ACS、Hypertensive emergency |
| **Nitroprusside** | 0.3-10 mcg/kg/min | 動靜脈均衡擴張；主動脈剝離、高血壓急症 |

### 高壓氧治療（Hyperbaric Oxygen Therapy）

| 適應症 | 機制 |
|--------|------|
| **CO 中毒** | 加速 CO-Hb 解離；縮短半衰期 |
| **氣壓傷/減壓病** | 減少氣泡體積；改善組織缺氧 |
| **壞死性軟組織感染** | 提高組織氧分壓；抑制厭氧菌 |
| **放射組織損傷** | 促進血管新生 |
| **傷口癒合延遲** | 刺激血管新生 |

## 重點摘要

- SCD 病因：80% 冠狀動脈疾病；EF ≤35% 為 ICD 候選
- **Brugada syndrome**：V1-3 ST elevation + RBBB pattern；東南亞常見
- **Long QT syndrome**：QTc >0.44s（心率 60）；Torsades de pointes
- **Shock 類型**：Hypovolemic / Cardiogenic / Obstructive / Distributive / Neurogenic
- **目標**：MAP >65 mmHg；Lactate <2 mmol/L；尿量 >0.5 mL/kg/hr
- **創傷大量輸血**：1:1:1（PRBC:FFP:Platelet）；TXA <3h；Permissive hypotension
- **過敏性反應**：IM Epinephrine 0.3-0.5 mg（大腿外側）
- **酸鹼失衡**：補償公式判讀；Anion gap 高 MA（MUDPILES）/正常（MUDPIES）
- **電擊能量**：VF 120-200J（biphasic）；同步 cardioversion 100-200J
- **首選升壓**：Norepinephrine（α1+β1）
- **心因性休克強心**：Dobutamine
