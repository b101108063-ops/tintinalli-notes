# Section 13：傳染病 (Infectious Diseases)

---

## Sexually Transmitted Infections (STIs)

### 一般原則

**診斷要點：**
- 病史需包含 sexual history（非judgemental 方式）
- Five Ps：Partners、Pregnancy prevention、Protection、Practices、Past history
- 多重感染常見（確診一個 STI 時應篩檢其他）
- 所有生育年齡女性需做 pregnancy test

**治療原則：**
- 疑似 gonorrhea/chlamydia → ED single-dose therapy
- 孕婦需 consultation
- 報告義務：Chlamydia、Gonorrhea、HIV、Syphilis
- 伴侣治療（最後一次性接觸前 60 天內）
- 建議 7-14 天後追蹤

### Chlamydia trachomatis

**臨床特徵：**
- 美國最常見 STI
- 女性常無症狀
- 男性：urethritis、epididymitis、proctitis、Reiter's syndrome
- 女性：asymptomatic cervicitis、vaginal discharge、intermenstrual bleeding、dysuria

**診斷：**
- NAAT：sensitivity 90%、specificity 99%
- 可使用 urine 或 vaginal specimens

**治療：**
| 藥物 | 劑量 |
|------|------|
| **Azithromycin** | 1g PO single dose |
| **Doxycycline** | 100mg PO bid × 7 days |
| **孕婦** | Azithromycin 或 amoxicillin |

### Neisseria gonorrhoeae

**臨床特徵：**
- 女性 80-90% 無症狀
- 男性 80-90% 有症狀（2 週內）
- 男性：dysuria + purulent penile discharge
- 女性：nonspecific lower abdominal discomfort + mucopurulent cervicitis
- Disseminated gonococcemia：petechial/pustular acral lesions、asymmetric arthralgias、tenosynovitis

**診斷：**
- NAAT（cervical、vaginal、urethral、urine）
- 男性：Gram stain（intracellular diplococci + PMNs）可確診

**治療：**
- **Dual therapy 建議**（因 resistance）：
  - Ceftriaxone 250mg IM + Azithromycin 1g PO 或
  - Cefixime 400mg PO + Azithromycin 1g PO
- **Fluoroquinolones**：不再建議（resistance）
- **孕婦**：ceftriaxone 或 azithromycin 2g PO
- **Disseminated**：Ceftriaxone 1g IV/IM q24h × 1-2 days → oral × 1 week

### Nongonococcal Urethritis

**原因：**
- C. trachomatis（最常見）
- U. urealyticum
- M. genitalium
- T. vaginalis
- HSV、adenovirus

**診斷：**
- Urethral specimen ≥5 WBCs/oil immersion field
- First-void urine 異常

**治療：**
- 同 chlamydia
- 如果持續 → T. vaginalis 治療（metronidazole 2g single dose）

### Trichomonas vaginalis

**臨床特徵：**
- Incubation：3-28 days
- 症狀：vulvar irritation、malodorous thin discharge、burning、pruritus、dysuria
- 男性通常無症狀
- 經期症狀可能加劇

**診斷：**
- Wet mount：motile parasites
- Culture（最準確但需 7 天）
- DNA probes/monoclonal antibodies

**治療：**
| 藥物 | 劑量 |
|------|------|
| **Metronidazole** | 2g PO single dose 或 500mg bid × 7d |
| **Tinidazole** | 2g PO single dose |
| **孕婦** | Metronidazole 2g single dose（僅有症狀時） |

**注意：** 服藥期間避免酒精

### Genital Ulcers 比較

| 疾病 | 病灶特徵 | 疼痛 | 淋巴結 |
|------|----------|------|--------|
| **Syphilis** | Indurated、clean base、painless | 無 | Firm、rubbery、nontender |
| **HSV** | Multiple vesicles → shallow ulcers | 有 | Tender bilateral |
| **Chancroid** | Painful、purulent、irregular | 有 | 50% suppurative buboes |
| **LGV** | Small shallow ulcer | 無 | Tender |
| **Granuloma inguinale** | Painless、beefy red、bleeding | 無 | 無 |

### Syphilis (Treponema pallidum)

**臨床分期：**
| 分期 | 表現 |
|------|------|
| **Primary** | Painless chancre（3-6 週自行癒合）、incubation ~21 days |
| **Secondary** | Rash（trunk、flexor surfaces、palms、soles）、lymphadenopathy、fever、sore throat |
| **Tertiary/Latent** | 3-20 年後：meningitis、dementia、tabes dorsalis、thoracic aneurysm |

**診斷：**
- Darkfield microscopy：直接看到 spirochete
- Nontreponemal tests（VDRL、RPR）：screening + monitor treatment response
- Treponemal tests（FTA-ABS）：confirmatory
- 需兩者配合確診

**治療：**
| 分期 | 治療 |
|------|------|
| **Primary/Secondary/Early latent** | Benzathine penicillin G 2.4M units IM single dose |
| **Latent/Late** | Benzathine penicillin G 2.4M units IM weekly × 3 weeks |
| **Penicillin allergy（非孕婦）** | Doxycycline 100mg bid × 14 days |

**Jarisch-Herxheimer reaction：** 治療後 24 小時內發生的急性發燒反應（常見於早期治療）

### Herpes Simplex Virus (HSV)

**臨床特徵：**
- 終身感染（HSV-1、HSV-2）
- 25% 美國人口有 serologic evidence
- Primary outbreak：prodrome（tingling、burning）→ vesicles → erosions → crusting
- 恢復通常 3 週內
- Recurrent outbreaks 較輕微

**診斷：**
- 臨床診斷為主
- Tzanck test：大 intranuclear inclusions
- Culture 或 PCR（vesicle fluid + base swab）

**治療：**
| 類型 | 藥物/劑量 |
|------|-----------|
| **First episode** | Acyclovir 400mg tid × 7-10d 或 famciclovir 250mg tid × 7-10d 或 valacyclovir 1g bid × 7-10d |
| **Recurrent（<9 episodes/年）** | Valacyclovir 1g daily × 5d |
| **Suppressive（>9 episodes/年）** | Valacyclovir 500mg daily |
| **Severe/Immunocompromised** | Acyclovir 5-10mg/kg IV q8h × 2-7d → oral |
| **孕婦** | Acyclovir 400mg tid × 7-10d |

### Chancroid (Haemophilus ducreyi)

**臨床特徵：**
- Painful erythematous papule → erosion/pustule
-  incubation：4-10 days
- Ulcers：sharp undermined margins、yellow-gray necrotic base
- "Kissing lesions"（autoinoculation）
- 50% painful suppurative lymphadenopathy

**診斷：**
- 臨床診斷（painful ulcer + lymphadenopathy）
- Culture（需要特殊培養基，sensitivity <80%）

**治療：**
| 藥物 | 劑量 |
|------|------|
| **Azithromycin** | 1g PO single dose |
| **Ceftriaxone** | 250mg IM single dose |
| **Ciprofloxacin** | 500mg bid × 3d |
| **Erythromycin** | 500mg qid × 7d |

### Lymphogranuloma Venereum (LGV)

**原因：** Chlamydia trachomatis L1、L2、L3

**臨床特徵：**
- Primary：painless chancre（很少被注意到）
- 1-3 週後：painful lymphadenopathy（buboes）
- 可能併發 proctocolitis、fistulas、strictures

**治療：**
- Doxycycline 100mg bid × 21 days
- 孕婦：erythromycin 500mg qid × 21 days

### Granuloma Inguinale (Donovanosis)

**原因：** Klebsiella granulomatis

**臨床特徵：**
- Endemic：Africa、Australia、India、New Guinea
- 美國罕見
- Painless、beefy red、bleeding ulcers
- 無淋巴結病變

**治療：**
| 藥物 | 劑量/療程 |
|------|-----------|
| **Doxycycline** | 100mg bid × 至少 3 週 |
| **Azithromycin** | 1g PO weekly × 至少 3 週 |
| **孕婦** | Erythromycin 500mg qid × 至少 3 週 |

### Bacterial Vaginosis

**臨床特徵：**
- Thin、gray、homogeneous discharge
- "Fishy" odor（尤其 with KOH）
- 正常 pH >4.5

**治療：**
| 藥物 | 劑量 |
|------|------|
| **Metronidazole** | 500mg bid × 7d 或 vaginal gel 5g daily × 5d |
| **Clindamycin** | Cream 2% 5g intravaginally qhs × 7d |
| **孕婦** | Metronidazole 500mg bid × 7d |

### STI 治療速查表

| STI | First-line | 替代/孕婦 |
|-----|-----------|-----------|
| **Chlamydia** | Azithromycin 1g PO 或 doxycycline 100mg bid × 7d | Azithromycin 1g PO 或 amoxicillin 500mg tid × 7d |
| **Gonorrhea** | Ceftriaxone 250mg IM + azithromycin 1g PO | Cefixime 400mg PO + azithromycin 1g PO |
| **Trichomoniasis** | Metronidazole 2g PO 或 tinidazole 2g PO | Metronidazole 2g PO（僅症狀者） |
| **Syphilis（早期）** | Benzathine penicillin G 2.4M units IM | Doxycycline 100mg bid × 14d |
| **HSV（首發）** | Acyclovir 400mg tid × 7-10d | Valacyclovir 1g bid × 7-10d |
| **Chancroid** | Azithromycin 1g PO 或 ceftriaxone 250mg IM | Erythromycin 500mg qid × 7d |
| **LGV** | Doxycycline 100mg bid × 21d | Erythromycin 500mg qid × 21d |
| **Bacterial vaginosis** | Metronidazole 500mg bid × 7d | Clindamycin cream |

### HIV Screening

- CDC 建議：13-64 歲常規篩檢
- ED 是 underused venue
- Rapid HIV testing 越來越普及
