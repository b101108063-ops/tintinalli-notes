# Section 11：婦產科

## Abnormal Uterine Bleeding

### 定義（FIGO）

| 術語 | 定義 |
|------|------|
| **Abnormal uterine bleeding** | 規律、量、頻率或持續時間異常 |
| **Heavy menstrual bleeding (HMB)** | 影響生活品質的過量月經出血 |
| **Amenorrhea** | >6 個月無月經 |
| **Intermenstrual bleeding** | 正常月經期間的出血 |
| **Postmenopausal bleeding** | 停經後任何出血 |

### 病因（PALM-COEIN）

| 結構性 | 非結構性 |
|--------|----------|
| **P**olyp | **C**oagulopathy |
| **A**denomyosis | **O**vulatory dysfunction |
| **L**eiomyoma | **E**ndometrial |
| **M**alignancy/hyperplasia | **I**atrogenic |
| | **N**ot classified |

### 年齡相關病因

| 年齡 | 常見原因 |
|------|----------|
| **青少年** | Anovulation（hypothalamic-pituitary axis 不成熟）、懷孕、凝血病 |
| **生育年齡** | 懷孕、anovulatory (PCOS)、結構性病變 |
| **更年期前期** | 子宮肌瘤、甲狀腺功能異常 |
| **停經後** | 萎縮性陰道炎（30%）、外源性激素使用（30%）、內膜病變/癌症（30%）|

### 評估

- **排除懷孕**：所有生育年齡女性
- **凝血病篩檢**：如有 heavy menstrual bleeding since menarche、postpartum hemorrhage、手術/牙科出血、family history
- **骨盆腔檢查**：speculum + bimanual exam

### 治療

| 情況 | 治療 |
|------|------|
| **Hemodynamically unstable** | 液體復甦、輸血、手術 |
| **Hemodynamically stable** | 賀爾蒙治療、取決於病因 |
| **anovulatory bleeding** | Progesterone 或 combined OCP |
| **結構性原因** | 手術處理 |

## Ectopic Pregnancy

### 危險因子

-  Prior ectopic pregnancy
-  Prior tubal surgery
-  PID
-  Current IUD use
-  Smoking
-  Infertility treatments

### 臨床表現

- **典型症狀**：停經 + 腹痛 + 陰道出血
- **也可能無症狀**
- **破裂風險**：可致命

### 診斷

| 檢查 | 發現 |
|------|------|
| **β-hCG** | >1500-2000 mIU/mL 時陰道超聲可見 gestational sac |
| **經陰道超聲** | 子宮內無 gestational sac + 附加發現（adnexal mass, free fluid）|
| **Culdocentesis** | 如有後穹窿積液 |

### 治療

| 類型 | 治療 |
|------|------|
| **Hemodynamically unstable/ruptured** | 緊急手術 |
| **Unruptured + 符合條件** | Methotrexate（single-dose IM）|
| **條件** | β-hCG <5000、、無胎心活動、無宮內妊娠 |

## First Trimester Complications

### 流產（Miscarriage）

| 類型 | 定義 |
|------|------|
| **Threatened** | 出血但宮頸閉合 |
| **Inevitable** | 出血 + 宮頸擴張 |
| **Incomplete** | 部分排出 |
| **Complete** | 完全排出 |
| **Missed** | 胚胎死亡但無排出 |

### 治療

- **Hemodynamically stable**：等待自然排出或藥物（misoprostol）
- **Hemodynamically unstable**：手術（ suction curettage）
- **Rh negative**：RhoGAM

---

## Second/Third Trimester Complications

### Preterm Labor

- **定義**：37 週前、子宮規律收縮 + 宮頸變化
- **治療**：
  - Tocolysis（如有指徵）
  - 皮質類固醇（24-34 週）
  - MgSO4（24-32 週神經保護）

### Placenta Previa

- **定義**：胎盤覆蓋宮頸內口
- **典型表現**：無痛性陰道出血
- **診斷**：超聲
- **治療**：臥床休息、如有嚴重出血則分娩

### Placental Abruption

- **定義**：胎盤在分娩前剝離
- **典型表現**：疼痛性子宮出血、子宮張力增加
- **併發症**：胎兒窘迫、凝血病變、DIC
- **治療**：取決於嚴重度和胎兒狀態

## Hypertensive Disorders of Pregnancy

### Preeclampsia

- **定義**：BP >140/90 + proteinuria 或 end-organ dysfunction（after 20 weeks）
- **嚴重特徵**：
  - BP >160/110
  - 血小板 <100,000
  - 肝酶升高
  - 腎功能異常
  - 肺水腫
  - 頭痛/視力改變

### Eclampsia

- **定義**：Preeclampsia + 癲癇發作
- **治療**：
  - **MgSO4**：4-6g IV bolus → 1-2g/h maintenance
  - 抗高血壓（labetalol、hydralazine）
  - 分娩

### HELLP Syndrome

- **定義**：溶血、肝酶升高、血小板減少
- **治療**：分娩（主要治療）

## Postpartum Emergencies

### 產後出血（Postpartum Hemorrhage）

- **定義**：陰道分娩 >500mL、剖腹產 >1000mL
- **原因（TONE）**：
  - **T**one（uterine atony）
  - **T**rauma（lacerations）
  - **T**issue（胎盤殘留）
  - **T**hrombin（凝血病變）

### 治療

| 原因 | 治療 |
|------|------|
| **Uterine atony** | 按摩、催產素、methylergonovine、carboprost |
| **Trauma** | 修復 |
| **Retained tissue** | 手術移除 |
| **Coagulopathy** | 輸血、FFP、cryoprecipitate |

### Amniotic Fluid Embolism

- **罕見但致命**
- **典型表現**：分娩中突然呼吸衰竭 + 循環衰竭 + DIC
- **治療**：支持性治療

## 新生兒急救

### Initial Assessment

- **Apgar score**：1、5、10 分鐘評估
- **關注**：呼吸、心率、膚色

### 常見緊急情況

| 情況 | 治療 |
|------|------|
| **呼吸窘迫** | O2、CPAP、必要時插管 |
| **窒息** | 初始復甦（suction、刺激）|
| **發紺** | 100% O2、超聲評估結構 |
| **早產** | 保溫、氧氣控制、維持血糖 |

## 骨盆腔發炎（PID）

### 診斷

- **最低標準（CDC）**：宮頸移動壓痛、子宮壓痛、附件壓痛
- **輔助標準**：發燒 >38.3°C、異常宮頸/陰道膿性分泌物、實驗室確診

### 治療

| 門診 | 住院 |
|------|------|
| Ceftriaxone 500mg IM + Doxycycline 100mg PO BID ×14d | IV cefoxitin 或 cefotetan + doxycycline |

### 併發症

- 輸卵管卵巢膿腫
- 不孕
- 慢性骨盆腔疼痛
- 子宮外孕風險增加

---

## Ovarian Torsion

- **典型表現**：突然單側下腹痛、噁心嘔吐
- **診斷**：臨床懷疑即手術（不可延誤）
- **超聲**：reduced/decreased blood flow
- **治療**：緊急手術解除扭轉

---

## 重點摘要

- **異常子宫出血**：排除懷孕、根據年齡考慮病因
- **子宫外孕**：破裂可致命；β-hCG + 超聲診斷；Methotrexate（符合條件時）
- **流產**：類型取決於宮頸狀態和超聲發現
- **Preeclampsia**：BP >140/90 + proteinuria after 20 weeks；嚴重特徵需積極處理
- **Eclampsia**：MgSO4 + 抗高血壓 + 分娩
- **HELLP**：溶血、肝酶升高、血小板減少；分娩
- **產後出血**：TONE 病因；按摩、催產素、必要時手術
- **PID**：CDC 最低標準確診；門診或住院治療
- **卵巢扭轉**：緊急手術
