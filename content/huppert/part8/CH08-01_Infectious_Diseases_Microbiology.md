# Microbiology

This chapter provides an overview of clinically important pathogens, adapted from *Medical Microbiology & Immunology: A Guide to Clinical Infectious Diseases, 15e* by Levinson, Chin-Hong, Joyce, Nussbaum, and Schwartz (McGraw-Hill Education, 2018).

## Overview of Clinically Important Pathogens

### Classification of Pathogens

| Type | Classification | Clinically Important Examples |
|------|---------------|------------------------------|
| **Bacteria** | Gram-positive: Cocci | Streptococcus spp., Staphylococcus spp. |
| | Gram-positive: Rods | Bacillus spp., Clostridium spp., Listeria spp. |
| | Gram-negative: Cocci | Neisseria spp. |
| | Gram-negative: Rods | Escherichia coli, Klebsiella spp., Pseudomonas aeruginosa, Hemophilus spp. |
| | Mycobacterium | Mycobacterium tuberculosis |
| | Spirochetes | Treponema pallidum (Syphilis), Borrelia spp., Leptospira spp. |
| | No cell wall | Mycoplasma spp. |
| **Viruses** | DNA viruses | Parvovirus, Papillomavirus (HPV), Adenovirus, Herpesvirus, Hepadnavirus (Hepatitis B) |
| | RNA viruses | Picornavirus, Hepevirus (Hepatitis E), Calicivirus (norovirus), Retrovirus (HIV), Orthomyxovirus (influenza), Coronavirus |
| **Fungi** | Yeasts | Candida spp., Cryptococcus spp. |
| | Dimorphic | Coccidioides spp., Histoplasma spp., Blastomyces spp., Paracoccidioides spp. |
| | Molds | Aspergillus spp., Mucor spp., Rhizopus spp. |

## Bacterial Infections

### Key Clinically Relevant Gram-Positive Bacteria

| Organism | Clinical Symptoms/Key Associations |
|----------|-----------------------------------|
| **Gram-Positive Cocci** | |
| Staphylococcus aureus | Skin/soft tissue infections, septic arthritis, osteomyelitis, pneumonia, endocarditis, meningitis. Toxin-mediated: Toxic shock syndrome, scalded skin syndrome, food poisoning. Can be methicillin-sensitive (MSSA) or methicillin-resistant (MRSA). Cover for MRSA empirically with IV vancomycin in hospitalized patients. |
| Staphylococcus epidermidis | Skin flora; can be a contaminant or a true pathogen. |
| Streptococcus pneumoniae | Meningitis, pneumonia, sinusitis, otitis media. |
| Streptococcus pyogenes (Group A) | Skin/soft tissue infection, pharyngitis. Toxin-mediated: Toxic shock syndrome, scarlet fever, necrotizing fasciitis. Immune-mediated: Rheumatic fever, post-strep glomerulonephritis. |
| Streptococcus agalactiae (Group B) | Meningitis, pneumonia, sepsis in newborns. |
| Streptococcus bovis | GI flora; can penetrate bloodstream and cause subacute bacterial endocarditis in colon cancer or IBD. |
| Enterococcus spp. | UTI, biliary infection, subacute bacterial endocarditis. E. faecium is often vancomycin-resistant (VRE). |
| **Gram-Positive Rods** | |
| Bacillus cereus (aerobic, spore-forming) | Preformed toxin contaminates rice; causes diarrhea. |
| Bacillus anthracis (aerobic, spore-forming) | Black painless ulcer, flu-like illness, lethal. |
| Clostridium tetani (anaerobic, spore-forming) | Wound infection; bacteria travels up neuron → spastic paralysis, trismus (lockjaw), risus sardonicus. |
| Clostridium botulinum (anaerobic, spore-forming) | Preformed toxin in canned food → flaccid paralysis. |
| Clostridium perfringens (anaerobic, spore-forming) | Wound infection → muscle necrosis; foodborne disease with watery diarrhea. |
| Clostridium difficile (anaerobic, spore-forming) | Common nosocomial infection; watery diarrhea or toxic megacolon. |
| Corynebacterium diphtheriae (non-filamentous, non-spore-forming) | Diphtheria. |
| Listeria monocytogenes (non-filamentous, non-spore-forming) | Poorly pasteurized milk/soft cheeses → gastroenteritis; meningitis in immunocompromised and those >50 years. |
| Actinomyces israelii (filamentous, non-spore-forming) | Normal GI/GU tract flora; can infect IUDs; causes abscesses. |
| Nocardia spp. (filamentous, non-spore-forming) | Pulmonary infections and brain abscess in immunocompromised hosts. |

### Key Clinically Relevant Gram-Negative Bacteria

| Organism | Clinical Symptoms/Key Associations |
|----------|-----------------------------------|
| **Gram-Negative Cocci** | |
| Neisseria gonorrhoeae | Sexually transmitted disease; vaginal discharge, PID, septic arthritis, dermatitis, tenosynovitis, migratory polyarthritis, Fitz-Hugh-Curtis syndrome. |
| Neisseria meningitidis | Meningitis. |
| **Gram-Negative Rods** | |
| Haemophilus influenzae (facultative, straight) | Encapsulated organism; pneumonia, meningitis, otitis media, epiglottitis. |
| Coxiella burnetii | Flu-like illness, pneumonia, granulomatous hepatitis, culture-negative endocarditis. |
| Legionella pneumophila | Grows in water tanks; severe pneumonia, GI symptoms, CNS symptoms, hyponatremia, diarrhea, encephalopathy. Dx: Urine antigen test, culture on legionella-specific media, or PCR. |
| Brucella | Brucellosis; cheese, goat/cow exposure → fever, osteomyelitis, or endocarditis. |
| Bartonella henselae | Cat scratch → lymph node enlargement, granulomas, fever. In HIV: bacillary angiomatosis. |
| Francisella | Tularemia; various forms including ulceroglandular, glandular, oropharyngeal, pneumonic, typhoidal. |
| Yersinia pestis | Plague. |
| Escherichia coli | UTI, diarrhea (ETEC; EHEC associated with HUS). |
| Enterobacter | UTI, nosocomial infections. |
| Klebsiella | UTI, pneumonia, liver abscess. |
| Shigella | Enterocolitis with bloody or watery diarrhea. |
| Salmonella | Enterocolitis with bloody or watery diarrhea; osteomyelitis in sickle cell disease; S. typhi causes typhoid fever. |
| Proteus | UTI. |
| Helicobacter pylori (facultative, curved) | Gastritis, peptic ulcer disease. |
| Campylobacter jejuni (curved) | Enterocolitis with bloody diarrhea. |
| Vibrio cholerae (curved) | Cholera, profuse watery diarrhea. |
| Pseudomonas aeruginosa (aerobic) | Pneumonia, UTI, nosocomial infections. |
| Bacteroides (anaerobic) | Peritonitis. |

### Key Clinically Relevant Other Bacteria

| Class | Organism | Clinical Symptoms/Key Associations |
|-------|----------|-----------------------------------|
| **Acid-Fast** | Mycobacterium tuberculosis (MTB) | Primary TB: Lung disease or extrapulmonary. Secondary TB: Reactivation in upper lobes, extrapulmonary TB. |
| | Mycobacterium avium-intracellulare (MAC) | Pneumonia with preexisting lung disease; disseminated infection in HIV with CD4 <50 cells/μL. |
| | Mycobacterium leprae | Skin and superficial nerves. |
| **Obligate Intracellular** | Rickettsia | Rocky Mountain spotted fever, typhus. |
| **Intracellular** | Chlamydia | Urethritis, psittacosis, lymphogranuloma venereum (LGV). |
| **Spirochetes** | Treponema pallidum | Syphilis: 1° (chancre), 2° (constitutional, rash), 3° (gummas, tabes dorsalis, Argyll Robertson pupil, aortitis). |
| | Leptospira interrogans | Leptospirosis; flu-like symptoms, jaundice, photophobia, conjunctivitis. |
| | Borrelia burgdorferi | Lyme disease; Ixodes tick. Early: erythema migrans. Early disseminated: Bell's palsy, cardiac involvement. Late: CNS, arthritis. |
| **No Cell Wall** | Mycoplasma | Community-acquired pneumonia. |

## Viral Infections

### Key Clinically Relevant DNA Viruses

| Virus | Family | Envelope | Capsid | DNA Structure | Clinical Associations |
|-------|--------|----------|--------|---------------|---------------------|
| Adenovirus | — | No | Icosahedral | DS linear | Pharyngitis, conjunctivitis, UTIs. |
| Parvovirus B19 | — | No | Icosahedral | SS linear | Aplastic crisis in sickle cell disease; adult RA-like symptoms; Fifth Disease in children (slapped cheek rash); hydrops fetalis. |
| Papillomavirus (HPV) | — | No | Icosahedral | DS circular | Warts; cervical/anal squamous cell carcinoma. |
| Herpesvirus | — | Yes | Icosahedral | DS linear | HSV-1: Oral lesions, cold sores, encephalitis. HSV-2: Genital herpes. VZV: Chickenpox (primary), shingles (reactivation). EBV: Mononucleosis. CMV: Mononucleosis-like syndrome (immunocompetent); encephalitis, retinitis, esophagitis (immunocompromised). HHV-6: Roseola. HHV-8: Kaposi's sarcoma. |
| Polyomavirus | — | No | Icosahedral | DS circular | JC Virus: PML in HIV. BK Virus: Transplant patients, kidney involvement. |
| Hepadnavirus | — | Yes | Icosahedral | DS partial circular | Hepatitis B virus (HBV). |
| Poxvirus | — | Yes | Complex | DS linear | Smallpox (eradicated); Molluscum contagiosum (umbilicated warts). |

### Key Clinically Relevant RNA Viruses

| Virus Family | Envelope | RNA Structure | Clinical Associations |
|--------------|----------|---------------|---------------------|
| Reovirus | No | DS linear | Rotavirus: Diarrhea. |
| Picornavirus | No | SS linear | Poliovirus: Aseptic meningitis, flaccid paralysis. Echovirus: Aseptic meningitis. Rhinovirus: Common cold. Coxsackievirus: Hand-foot-mouth disease, myocarditis, encephalitis. Hepatitis A Virus (HAV). |
| Calicivirus | No | SS linear | Norovirus: Viral gastroenteritis. |
| Hepevirus | No | SS linear | Hepatitis E Virus (HEV). |
| Flaviviruses | Yes | SS linear | Yellow fever (Aedes mosquito); Dengue (breakbone fever); West Nile virus (encephalitis, flaccid paralysis); Hepatitis C Virus (HCV). |
| Togaviruses | Yes | SS linear | Rubella; Eastern/Western Equine Encephalitis. |
| Retroviruses | Yes | SS linear | HIV; HTLV (T-cell leukemia). |
| Coronaviruses | Yes | Helical SS linear | SARS-CoV2 (COVID-19): Fever, respiratory compromise, leukopenia. |
| Orthomyxoviruses | Yes | Helical SS linear | Influenza virus. |
| Paramyxoviruses | Yes | Helical SS linear | Parainfluenza (croup); RSV (bronchiolitis); Measles (cough, conjunctivitis, rash); Mumps (parotitis, orchitis, pancreatitis). |
| Rhabdoviruses | Yes | Helical SS linear | Rabies: Bats, mammals; retrograde travel up nerves; fever, agitation, photophobia, hydrophobia; paralysis and death. |
| Filoviruses | Yes | Helical SS linear | Ebola hemorrhagic fever: Often fatal; febrile syndrome, rash, mucosal bleeding, multiorgan failure. |
| Arenaviruses | Yes | Helical SS circular | LCMV: Rodent-borne; meningitis. Lassa fever. |
| Bunyaviruses | Yes | Helical SS circular | Hantavirus: Rodent feces; hemorrhagic fever, pneumonia, respiratory failure. |
| Deltavirus | Yes | Uncertain SS circular | Hepatitis D virus (HDV): Requires HBV for infection. |

## Fungal Infections

### Key Clinically Relevant Systemic Fungal Infections

| Genus | Morphology in Tissue | Geographic Distribution | Clinical Information |
|-------|----------------------|------------------------|---------------------|
| Coccidioides | Spherule | Southwestern US (CA, AZ), Latin America | Pulmonary infection; can disseminate to bone, joints, meninges, skin. Dx: Serology, culture, cocci antigen. Tx: Fluconazole. |
| Histoplasma | Yeasts inside macrophages | Mississippi/Ohio River Valley | Cavitary lung lesions, granulomas in liver/spleen, skin, bones, prostate. Dx: Histo urine antigen, serology. Tx: Itraconazole; amphotericin B if severe. |
| Blastomyces | Yeasts with single broad-based buds | Southeastern US, Central America | Inflammatory lung lesions, ulcerated skin lesions. Dx: Tissue with broad-based buds, antigen testing. Tx: Itraconazole; amphotericin B if severe. |
| Paracoccidioides | Yeasts with multiple buds | Latin America | Pneumonia, calcified lung mass, oral mucosal lesions. Dx: Culture. Tx: Itraconazole; amphotericin B if severe. |

### Key Clinically Relevant Opportunistic Fungal Infections

| Genus | Clinical Features | Treatment |
|-------|------------------|-----------|
| **Candida** | Yeast forms hyphae and pseudohyphae worldwide. Vaginal: "yeast infection" with cottage cheese-like discharge. UTI: True UTI rare, common in catheterized patients. Mouth: "thrush" with white plaques. Esophagitis: White plaques that scrape off, odynophagia. Cutaneous: Erythematous patches with satellite lesions. Bacteremia: Never a contaminant. Dx: KOH prep, germ tube test. | Vaginal: Topical azole cream. Thrush: Clotrimazole, nystatin, fluconazole. Esophagitis: Fluconazole. Cutaneous: Nystatin powder. Bacteremia: IV echinocandin. |
| **Cryptococcus** | Yeast with large capsule worldwide. Pigeon droppings → inhale → hematogenous spread. Risk: HIV (CD4 <50), immunosuppression. Pulmonary disease, meningitis with elevated opening pressure. Dx: Serum CrAg, LP. | Fluconazole 6–12 months (longer if HIV). Amphotericin B + Flucytosine for meningitis induction. |
| **Aspergillus** | Mold with narrow-angle branching, septate hyphae worldwide. ABPA: Type I hypersensitivity, asthma, eosinophilia, +IgE. Aspergilloma: Fungus ball in preexisting cavity. Invasive: Invades lungs, sinuses, brain in immunocompromised. Dx: Serum galactomannan, Beta-D-glucan. | ABPA: Steroids, azoles. Aspergilloma: Surgical removal ± azoles. Invasive: Voriconazole. |
| **Mucor & Rhizopus** | Mold with 90-degree angle branching, non-septate hyphae worldwide. Invades blood vessels, causes necrotic lesions. Look for "reverse halo sign" on chest CT. | Liposomal amphotericin B, surgical debridement. |

## Parasitic Infections

### Key Clinically Relevant Protozoa

| Organism | Transmission | Clinical Features |
|----------|--------------|-------------------|
| **GI Infections** | | |
| Giardia lamblia | Ingestion of cysts in water (camping, hiking) or person-to-person | Giardiasis: Foul-smelling fatty diarrhea. |
| Entamoeba histolytica | Ingestion of cysts in water in tropical locations | Bloody diarrhea, ulcers, liver abscess, RUQ pain. |
| Cryptosporidium parvum | Ingestion of cysts in water | Diarrhea, severe in immunocompromised. |
| **CNS Infections** | | |
| Toxoplasma gondii | Ingestion of cysts in poorly cooked pork, cat feces | Rim-enhancing brain lesion(s). Maternal-fetal: Chorioretinitis, hydrocephalus, intracranial calcification. |
| Naegleria fowleri | Swimming in fresh water lake | Rapidly fatal meningoencephalitis. |
| Trypanosoma brucei (gambiense/rhodesiense) | Tsetse fly bite | African sleeping sickness: Enlarged lymph nodes (Winterbottom's sign), recurring fever, personality changes, somnolence, coma. |
| **Hematologic Infections** | | |
| Plasmodium (P. vivax/ovale, P. falciparum, P. malariae) | Mosquito (Anopheles) | Malaria: Fever, anemia, splenomegaly. |
| Babesia | Ixodes tick (same as B. burgdorferi) | Babesiosis: Fever and hemolytic anemia, risk if asplenic. Most common transfusion-related infection. |
| **Visceral Infections** | | |
| Trypanosoma cruzi | Reduviid bug (painless bite) in South America | Chagas' disease: Dilated cardiomyopathy, megacolon, megaesophagus, unilateral eye swelling (Romaña's sign). |
| Leishmania donovani | Sandfly | Visceral leishmaniasis: Lesions on face/skin, round belly, spiking fevers, hepatosplenomegaly, pancytopenia. |
| **Sexually Transmitted Disease** | | |
| Trichomonas vaginalis | Sexual transmission | Vaginitis: Foul-smelling frothy yellow/green discharge. |

### Key Clinically Relevant Helminthic Infections

| Organism | Transmission | Clinical Features |
|----------|--------------|-------------------|
| **Nematodes (Roundworms) – Intestinal** | | |
| Enterobius vermicularis (Pinworm) | Food contaminated with eggs | Intestinal infection with perianal itching in children (Scotch tape test). |
| Ascaris lumbricoides (Giant white roundworm) | Fecal-oral | Eggs hatch in duodenum; migrate through lungs (pneumonitis); mature in GI tract; can obstruct biliary or GI tract. |
| Strongyloides stercoralis | Larvae in soil penetrate skin | Intestinal: Vomiting, diarrhea, anemia. Disseminated: Pulmonary symptoms (cough to ARDS), cutaneous (rash), GI symptoms. |
| Ancylostoma duodenale / Necator americanus (Hookworm) | Larvae in feces penetrate skin | Microcytic anemia by sucking blood from intestinal walls. |
| **Nematodes (Roundworms) – Tissue** | | |
| Dracunculus medinensis | Drinking water with copepods | Migrate down leg (painful), skin inflammation/ulcers. |
| Onchocerca volvulus | Female blackfly bite | Hyperpigmented skin, river blindness. |
| Loa Loa | Deer fly, horse fly, mango fly | Swelling in skin, worm in conjunctiva. |
| Wuchereria bancrofti | Female mosquito | Blocks lymphatic vessels; elephantitis. |
| Toxocara canis | Food contaminated with eggs | Visceral larva migrans. |
| **Cestodes (Tapeworms)** | | |
| Taenia solium | Ingestion of larvae in undercooked pork (Central America, Southeast Asia) | Larvae: Intestinal infection. Eggs: Neurocysticercosis → brain cysts + seizures. |
| Diphyllobothrium latum | Consumption of fresh water or raw fish (Scandinavia, former Soviet Union, Japan) | Usually asymptomatic. Classic: Vitamin B12 deficiency and megaloblastic anemia. |
| Echinococcus granulosus | Ingestion of eggs in dog feces | Cysts in liver: Anaphylaxis if antigens released (preinject ethanol before removal). |
| **Trematodes (Flukes)** | | |
| Schistosoma (S. haematobium, S. mansoni) | Snails are host; penetrate skin → liver → bladder | "Swimmer's itch," fever. Eggs lodged in liver/spleen → granulomas and fibrosis. S. haematobium: Hematuria, bladder cancer. S. mansoni: Non-cirrhotic portal HTN. |
| Clonorchis sinensis (Chinese Liver Fluke) | Undercooked fish, Southeast Asia | Biliary tract inflammation, recurrent pyogenic cholangitis, pigmented gallstones. Associated with cholangiocarcinoma. |
| Paragonimus westermani | Undercooked crab meat | Lung inflammation. |
